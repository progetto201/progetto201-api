{\footnotesize\ttfamily valid\+\_\+times (\begin{DoxyParamCaption}\item[{}]{\$t\+\_\+conn\+\_\+res }\end{DoxyParamCaption})}



Checks if the request timestamps are valid. 

The function collect the mintime and maxtime timestamps, checks if they are numbers.

If they are numbers their values must be equal to zero or the mintime value has to be lower than the maxtime value.

In these cases, the \mbox{\hyperlink{date_8php_a4d0c0508d0cfd38c221f532d560cd5d3}{set\+\_\+times()}} function is called to set the time in the DB.

Otherwise an error is collected in the \$action\+\_\+res array
