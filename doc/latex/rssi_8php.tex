\doxysubsection{Detailed Description}
{\bfseries{R\+S\+SI}}\+: manages R\+S\+SI of the nodes. 

Client perspective\+:
\begin{DoxyItemize}
\item {\bfseries{G\+ET}} M\+E\+T\+H\+OD\+: gets R\+S\+SI, type description and location of the nodes.
\end{DoxyItemize}

The script tries to connect to the database and then checks the request method.

If the method is G\+ET \mbox{\hyperlink{rssi_8php_afe154ce4a8f57881bea6db3b969cad7d}{get\+\_\+rssi()}} is called.
